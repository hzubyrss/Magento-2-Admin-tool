Magento 2 Admin Tools

Simplify your Magento 2 administration with powerful tools for reindexing, deployment, and system commands.

Get Started View Features

Overview

The Magento 2 Admin Tools extension provides a user-friendly interface for performing common administrative tasks directly from your Magento admin panel. This extension eliminates the need to access the server via SSH or command line for routine operations.

Easy Reindexing

Reindex individual or all indexers with a single click from the admin panel.

System Commands

Run setup:upgrade, setup:di:compile, and static content deployment without command line access.

Mode Switching

Easily switch between developer and production modes with a single click.

Installation

1

Create Directory Structure

Create the following directory structure in your Magento installation:

app/code/Storeteam/AdminTools/
2

Copy Files

Copy all extension files into the directory structure, maintaining the file hierarchy.

3

Run Installation Commands

Execute the following commands from your Magento root directory:

php bin/magento module:enable Storeteam_AdminTools
php bin/magento setup:upgrade
php bin/magento setup:di:compile
php bin/magento setup:static-content:deploy -f
php bin/magento cache:flush
4

Verify Installation

Log in to your Magento admin panel and navigate to System > Admin Tools to verify the installation was successful.

Features

Index Management

The Index Management feature provides a user-friendly interface to manage and reindex your Magento indexes directly from the admin panel.

  • View all available indexers in one place
  • Reindex individual indexers with a single click
  • Reindex all indexers at once
  • View detailed output of reindexing operations
  • Confirmation dialogs to prevent accidental reindexing

System Commands

Execute common Magento commands directly from the admin panel without needing command line access.

  • Run setup:upgrade to update database schema
  • Run setup:di:compile for code generation
  • Deploy static content for specific locales
  • View detailed command output
  • Confirmation dialogs for potentially time-consuming operations

Deployment Modes

Easily switch between Magento's deployment modes directly from the admin panel.

  • View current deployment mode
  • Switch to developer mode for development
  • Switch to production mode for live sites
  • Confirmation dialogs before mode switching

Security Features

The extension is built with security in mind, providing several features to ensure safe operation.

  • Role-based access control using Magento's ACL system
  • Optional confirmation dialogs before executing commands
  • Secure command execution using Magento's Shell class
  • Module can be disabled completely via configuration

Usage

Accessing Admin Tools

After installation, you can access the Admin Tools from the main menu:

  1. Log in to your Magento admin panel
  2. Navigate to System > Admin Tools
  3. Choose either Index Management or System Commands

Using Index Management

To reindex your Magento store:

  1. Navigate to System > Admin Tools > Index Management
  2. To reindex all indexes at once, click the Reindex All button
  3. To reindex a specific index, find it in the table and click the Reindex button in its row
  4. Confirm the action if prompted
  5. View the reindexing results in the result section that appears

Using System Commands

To run system commands:

  1. Navigate to System > Admin Tools > System Commands
  2. Choose the command you want to execute:
    • Click Run setup:upgrade to update database schema
    • Click Run setup:di:compile to generate code
    • Enter a locale code and click Run setup:static-content:deploy
    • Click either Developer Mode or Production Mode to switch modes
  3. Confirm the action if prompted
  4. View the command output in the result section that appears

Configuration

You can configure the Admin Tools extension under Stores > Configuration > Store Team > Admin Tools.

General Settings

  • Enable Module: Enable or disable the entire module
  • PHP Path: Specify a custom PHP binary path if your server requires it

Security Settings

  • Confirm Command Execution: Enable or disable confirmation dialogs before executing commands

Access Control

To control which admin users can access the Admin Tools:

  1. Navigate to System > User Roles
  2. Edit the role you want to modify
  3. In the Role Resources tree, find System > Admin Tools
  4. Check or uncheck the permissions as needed
  5. Save the role

Frequently Asked Questions

Is it safe to run these commands from the admin panel?

Yes, the extension uses Magento's built-in Shell class to execute commands securely. Additionally, access to these features is restricted by Magento's ACL system, so only authorized administrators can use them.

Will reindexing affect my store's performance?

Reindexing can be resource-intensive, especially on large stores. It's recommended to run full reindexing operations during off-peak hours. The extension provides individual indexer reindexing, so you can choose to reindex only what's necessary.

What happens if a command fails?

If a command fails, the extension will display the error message and any output from the command. This information can help you diagnose and fix the issue. The extension handles errors gracefully and won't leave your store in an inconsistent state.

Can I use this extension on a production store?

Yes, the extension is designed to be used on production stores. However, we recommend testing it in a development environment first to ensure it works correctly with your specific Magento setup.

Does this extension work with Magento Commerce (Enterprise Edition)?

Yes, the extension is compatible with both Magento Open Source (Community Edition) and Magento Commerce (Enterprise Edition).
